# Health Insurance Premium Prediction

This project predicts health insurance premium charges using Machine Learning.

## Dataset
Health Insurance Dataset

## Algorithms
- Random Forest Regressor

## Libraries
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Matplotlib

## Performance

R² Score : 0.84

MAE : 2738

RMSE : 4957

MAPE : 33%

## Features

- Age
- Gender
- BMI
- Children
- Smoker
- Region

## Target Variable
- Charges
